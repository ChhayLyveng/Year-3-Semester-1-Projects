# Statistics Project: Student Housing Costs 

## Project Overview
A primary research project involving the collection and statistical analysis of student survey data regarding monthly housing expenses.

## Statistical Methods Applied
* **Data Collection:** Designing and executing a survey for primary data.
* **Descriptive Statistics:** Analyzing mean, median, and variance of housing costs.
* **Inference:** Calculating **Point Estimators** and **Confidence Intervals**.
* **Hypothesis Testing:** Testing assumptions about the population based on survey samples.
* **Visualization:** Plotting cost distributions to identify spending trends among students.

## Tech Stack
* **Python**, **NumPy**, **Pandas**, **Matplotlib**, **Seaborn**
