# Programming for Data Science: Zomato Analysis 

## Project Overview
An analysis of secondary data sourced from **Kaggle** regarding Zomato restaurant listings. The goal was to identify factors that contribute to restaurant success and rating trends.

## Analysis Workflow
* **Data Wrangling:** Cleaning the Kaggle dataset for consistency.
* **Feature Exploration:** Analyzing price ranges, cuisines, and location data.
* **Visualization:** Using Seaborn and Matplotlib to map out restaurant density and rating distributions.

## Tech Stack
* **Python**, **Pandas**, **Matplotlib**, **Seaborn**
