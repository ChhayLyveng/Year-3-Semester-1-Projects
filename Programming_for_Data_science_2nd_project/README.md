
# Programming for Data Science: Quotes Web Scraping 

## Project Overview
This project focuses on automated data collection and feature engineering using web scraping techniques to analyze "Quotes" data.

## Technical Implementation
* **Web Scraping:** Utilizing `BeautifulSoup4` and `Requests` to extract data from web pages.
* **Ethical Scraping:** Implementing `Time` delays to manage request frequency.
* **Feature Engineering:** Creating new variables from scraped text to enhance analysis.
* **Data Analysis:** Visualizing common themes and author trends.

## Tech Stack
* **Libraries:** BeautifulSoup4, Requests, Pandas, Matplotlib, Seaborn
